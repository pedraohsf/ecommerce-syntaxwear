# 🛒 SyntaxWear | E-commerce para Desenvolvedores

Bem-vindo ao repositório do **SyntaxWear**, uma loja virtual de vestuário e acessórios criada especialmente para desenvolvedores!

Este projeto foi estruturado utilizando as melhores práticas de organização de arquivos de desenvolvimento web front-end.

---

## 📂 Estrutura do Projeto

O projeto está organizado da seguinte forma:

```text
/
├── index.html (Homepage principal)
├── css/
│   ├── base.css (Estilos globais, variáveis CSS e resets)
│   ├── layout.css (Helpers de alinhamento, grid e espaçamentos)
│   └── components/ (Estilos individuais de componentes)
│       ├── header.css
│       ├── panel.css
│       ├── hero.css
│       ├── product-card.css
│       ├── product-grid.css
│       ├── newsletter.css
│       └── footer.css
├── images/ (Imagens organizadas por categoria)
│   ├── logo/
│   ├── banners/
│   └── products/
├── icons/ (Arquivos de ícones SVG)
├── fonts/ (Fontes locais do projeto)
├── favicons/ (Ícones de aba de navegador)
└── docs/ (Documentações e anotações do projeto)
```

## 🚀 Como Executar o Projeto Localmente

Como este é um projeto estático em HTML e CSS, você pode executá-lo de forma muito simples:

1. Clone este repositório.
2. Abra a pasta do projeto no seu editor de código favorito (recomendamos o **VS Code**).
3. Abra o arquivo `index.html` diretamente no seu navegador, ou utilize uma extensão como a **Live Server** no VS Code para ter recarregamento automático ao salvar alterações.
